# GitHub Collaboration Project

This project is built using **HTML & CSS**.

## Structure
- `index.html` – Main layout (Nav, Home, Footer)
- `index.css` – Global styles

## Sections Responsibility
- **Nav & Footer** → Maxamed
- **Home & Main Sections** → Teammate

## Workflow
1. Each section is developed in a separate branch
2. Changes are merged via Pull Requests
3. GitHub handles merging automatically

## How to run
Just open `index.html` in the browser.
